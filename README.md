# jls-sof
翻译、整理、总结了Stack Overflow
关于Java 的优秀QA。

### 范围
Java体系过于庞大，本博客系列仅选择
- 包含Java 语言规范和 java.lang包下的知识点
- 但是排除了内存模型和多线程相关的内容

### 为什么我要看你的博客而不是直接在SOF上找？
- 很多内容和它的赞数不符合，一些没太大营养
  的QA被顶的很前面，很过高质量的QA排在很后面

- 很多重复的内容

- QA很杂乱，直接阅读效率低

### 目录
为了保证博客的可维护性，没有给博客编号，
原则上不同的章节是没有顺序要求的，大家
可以自行选择

### 建议
有什么建议可以提issue，发现一些错漏的地方
欢迎直接pull request
